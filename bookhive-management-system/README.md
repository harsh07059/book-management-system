# 📚 BookHive — Book Management System

A modern and responsive Book Management System built using React.js, Vite, Tailwind CSS, and MockAPI.  
This application allows users to manage books with full CRUD functionality including adding, editing, deleting, searching, and filtering books.

# ✨ Features

✅ View all books  
✅ Add new books  
✅ Edit existing books  
✅ Delete books  
✅ Search books by title or author  
✅ Filter books by genre  
✅ Responsive modern UI  
✅ API integration using MockAPI  
✅ Real-time CRUD updates  
✅ Loading states and error handling

# 📦 Installation & Setup Instructions

## 1️⃣ Clone Repository

```bash
git clone https://github.com/harsh07059/book-management-system.git
```

---

## 2️⃣ Navigate Into Project Folder

```bash
cd book-management-system
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Start Development Server

```bash
npm run dev
```
